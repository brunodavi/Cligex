# Cligex

command to test regex by terminal

```
Usage:
        cligex -t, --text | -s, --search | [ -r, --replace ] | [ -h, --help ]

Capture matches/replace by regex

-t|--text
        Text to search

-s|--search
        Search in regex

-r|--replace
        Replace search founding in text

-h|--help
        Show this message
```
